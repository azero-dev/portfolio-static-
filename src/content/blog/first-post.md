# Building the Future with Astro

Welcome to the first post of my cyberpunk-inspired portfolio blog. This site is built with **Astro**, **React**, and **Tailwind CSS**, featuring a Minimal Cyberpunk design theme.

## Why Astro?

Astro offers the perfect blend of performance and flexibility. By default, Astro ships zero JavaScript to the browser, resulting in incredibly fast load times. When you need interactivity, you can drop in React components that hydrate only when necessary.

## Cyberpunk Aesthetic

The Minimal Cyberpunk theme combines:

- Dark backgrounds (#111111)
- Neon orange accents (#ff7733)
- Monospace typography (Share Tech Mono)
- Clean, spacious layouts

## Key Features

1. **Responsive Navigation** - Desktop menu with mobile overlay
2. **Theme Toggle** - Light/dark mode with next-themes
3. **Project Showcase** - Filterable projects with tabs
4. **Blog System** - What you're reading right now!

## Upcoming Features

- ActivityPub integration with Fedify
- Full-text search with Pagefind
- RSS feeds and sitemap
- Contact form with validation

Stay tuned for more updates as this portfolio evolves. The future is bright... and orange.
